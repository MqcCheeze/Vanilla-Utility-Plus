# Vanilla Utility Plus
## A little utility mod I have created to learn how to mod Minecraft and to help myself in my survival world.

## What it adds?
## *.* (dot) Commands
- .overworld
- .nether
- .time
- .nyan <message>
- .track <add | remove | list> [playerName]

## */* (slash) Commands
- /pos <nether | overworld>
